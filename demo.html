如何启动实际应用

方法1：Docker部署（推荐）
# 1. 启动所有服务
docker-compose up -d

# 2. 初始化数据库
docker-compose exec web flask init-db
docker-compose exec web flask create-sample-data

# 3. 访问应用
http://localhost:5000  # 主网站
http://localhost:5000/admin  # 管理后台
http://localhost:8080  # 数据库管理(Adminer)
方法2：本地开发
# 1. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 设置环境变量
export FLASK_APP=app.py
export FLASK_ENV=development

# 4. 初始化数据库并启动
flask init-db
flask create-sample-data
python app.py
